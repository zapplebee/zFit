zFit

A jQuery plugin to vertically center and shrink to fit contents of an element.

Let each element find its largest size text without regard for the selection

Use $(".resize1").zFit(); to fit all text independently within the selection. Also, best option if a single item is in the selector (i.e. $("#myElement")).

Or match the sizes of all selected elements

Use $(".resize2").zFit(true); to match all text to the size of the smallest text. Great for buttons of variable length or links.
